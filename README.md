# YK070201E Portfolio

개발자 포트폴리오 웹사이트입니다. React와 TypeScript를 사용하여 제작되었으며, 다크/라이트 테마 전환 기능을 포함하고 있습니다.

## 🚀 프로젝트 개요

이 프로젝트는 한봄고등학교 학생의 개발 포트폴리오를 보여주는 웹사이트입니다. UI/UX 디자인, 프론트엔드 개발, FIVEM 스크립트 제작 등 다양한 프로젝트 경험을 담고 있습니다.

## ✨ 주요 기능

- **반응형 디자인**: 모바일과 데스크톱에서 최적화된 사용자 경험
- **테마 전환**: 다크/라이트 모드 지원
- **포트폴리오 갤러리**: 이미지와 동영상을 포함한 프로젝트 전시
- **개인 정보 섹션**: 학력, 교육 활동, 자격증, 수상 경력 등

## 🛠️ 기술 스택

- **Frontend**: React 19.1.0, TypeScript 5.8.3
- **Build Tool**: Vite 7.0.4
- **Styling**: CSS3
- **Linting**: ESLint 9.30.1
- **Package Manager**: npm

## 🚀 시작하기

### 필수 요구사항

- Node.js 18.0.0 이상
- npm 9.0.0 이상

### 설치 및 실행

1. **저장소 클론**
   ```bash
   git clone [repository-url]
   cd yk070201e_portfolio
   ```

2. **의존성 설치**
   ```bash
   npm install
   ```

3. **개발 서버 실행**
   ```bash
   npm run dev
   ```

4. **빌드**
   ```bash
   npm run build
   ```

5. **프리뷰**
   ```bash
   npm run preview
   ```
